Dual-Boot Documentation

Welcome to the project documentation. This guide will help you with the process of dual-booting Windows and Ubuntu, covering everything from BIOS configuration to setting up Ubuntu. Below you'll find detailed information about each component of the project.

Overview

Creating a Bootable USB with Rufus

This section will guide you through the process of creating a bootable USB drive using Rufus. You will use this USB drive to install Ubuntu alongside Windows.

BIOS Configuration

In this section, you will learn how to configure your BIOS settings to prepare your system for dual-booting. This includes disabling Secure Boot, changing SATA operation mode, and applying the necessary changes.

Installing Ubuntu

Here, you will find detailed instructions on how to install Ubuntu on your system. This includes partitioning your hard drive, choosing the installation options, and completing the installation process.

Ubuntu Setup

After installing Ubuntu, there are several setup steps to get your system running smoothly. This section covers initial configurations, installing updates, and setting up essential software.

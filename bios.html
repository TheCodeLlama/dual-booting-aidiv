This guide will help you configure the BIOS settings on your Dell Precision 3480.

Step 1: Access the BIOS

  1. Restart your computer.
  2. Press F2 repeatedly to enter the BIOS setup.

Step 2: Overview Page

You should see an overview page with your system information.

Step 3: Disable Secure Boot

  1. Navigate to the Boot Configuration tab.
  2. Locate the Secure Boot section.
  3. Click on Secure Boot to disable it.
  4. Confirm the changes by selecting Yes.

Step 4: Change SATA Operation

  1. Navigate to the Storage tab.
  2. Select SATA/NVMe Operation.
  3. Change the setting from RAID On to AHCI/NVMe.
  4. Confirm the changes by selecting Yes.

Step 5: Apply Changes

After making all changes, click Apply Changes.

Step 6: Exit BIOS

  1. Click Exit to leave the BIOS setup.
  2. This will Restart your computer.

Step 7: Select Boot Device

  1. Press F12 repeatedly to enter the BIOS setup.
  2. Navigate to the One-Time Boot Settings tab.
  3. Select UEFI USB Flash Memory as the boot device.
  4. Double Click on the UEFI USB Flash Memory (you can also press Enter once it's selected).

Your BIOS configuration should now be complete.
